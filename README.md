# telegram-bot-3xui
Telegram‑бот для управления VPN‑ключами через панель 3x‑ui и интеграция с панелью 3x-ui.

## О проекте
Код написан с использованием AI‑ассистента, доработан вручную и адаптирован под реальные задачи.  
Цель проекта — отработка навыков DevOps и интеграции Telegram‑бота с панелью 3x‑ui.

Бот для интеграции с панелью 3x-ui. 
Функции: 
- Создание и продление VPN-ключей
- Пополнение трафика для лимитных тарифов
- Поддержка тарифных планов (лимитные/безлимитные)
- Уведомления и подтверждение оплаты администратором
- Интеграция с UIAPI (Python)

## Стек
- Python v3.12.6
- requests
- JSON-хранилища для пользователей и тарифов

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить переменные окружения: - `BOT_TOKEN` - `UI_BASE_URL` - `UI_USERNAME` - `UI_PASSWORD`
4. Запустить: `python bot.py`

## Пример использования
- Пользователь: `/add_key mylogin`
- Админ: подтверждает оплату и выдаёт ключ
